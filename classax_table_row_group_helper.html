axTableRowGroupHelper::before ( $  content,
$  replace_callback = null 
)

Add column(s) before row content.

You may pass a callback to transform those cells and/or to change replace parameters. The callback must take a first parameter the cell(s) you are inserting (as array if multiple cells) and the rows to be inserted as second parameter. The second parameter will be provided during the array construction.

E.G > // add a message with a provided parameter before each table row > $table->body->before("Id : %d", 'function ($cell, $values) { return sprintf($cell[0], $values["id"]); }');
